## [0.1.0] - 2026-03-04

### Added

- Single-file document analysis for PDF, DOCX, and PPTX via Docling
- Directory profiling with aggregate statistics and filetype distribution
- SQLite caching with size/mtime invalidation
- Rich terminal output and JSON output modes
- Per-file detail tables in directory mode (`--detail`)
- Annotated page image export (`--save-images`)
- Verbose logging (`--verbose`)
